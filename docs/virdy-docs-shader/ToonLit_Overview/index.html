ToonLit - 개요


ToonLit 셰이더는 VIRDY 전용 카툰 셰이더로, PBR + NPR의 장점을 살린 고성능 커스텀 셰이더입니다.

ToonLit 원본 셰이더에 대한 전체 소스코드 접근은 VIRDY 팀과의 별도 계약이 이루어진 소수의 크리에이터, 아티스트에게만 배포되고 있습니다.

현재는 ToonLit의 경량화 버전을 배포하여 최소한의 기능을 사용할 수 있도록 하고 있습니다.
다만, 실제 VIRDY 내에 접속시 ToonLit 원본 셰이더로 자동 컨버팅되어 적용됩니다.

세팅 방법​

ToonLit 셰이더는 Unity Renderer에 Add Feature를 통해 ToonLit 셰이더를 추가하여야 합니다.
