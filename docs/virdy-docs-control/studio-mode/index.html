스튜디오 모드

tip

스튜디오 모드는 매니저 모드와 비슷하지만, 기능이 확장된 버전입니다.
콘텐츠 진행을 위한 여러명의 카메라맨이 필요할 경우 적합한 모드입니다.


스튜디오 모드에서는 고정 카메라를 전부 실시간으로 확인 할 수 있습니다.

다만, PC 환경에 따라 프레임이 안나올 수 있습니다.
카메라 조작 탭에서 모니터링 해상도 배율 및 프레임 제한 을 설정하여 프레임을 확보하여 주세요.

스튜디오 모드에서는 Sub Screen 에 자신의 고정 카메라가 아닌, 다른 유저의 카메라를 할당할 수 있습니다.


카메라 세팅 탭의 프리캠을 제외한 고정 카메라에서 채널 링크를 누르면 자신을 제외한 다른 유저의 채널이 보입니다.

  • '닉네임'은 해당 유저의 Main Screen 을,
  • '닉네임#0'은 해당 유저의 Manager Screen 을,
  • '닉네임#n'은 해당 유저의 각 Sub Screen 을 뜻합니다.

ex. 3번 카메라 세팅에서 채널 링크를 '테스트#7'로 설정시,
테스트 유저의 7번 카메라 시점을 자신의 고정 카메라 3번에 설정합니다.

caution

매니저 모드에서도 채널 링크 기능을 사용할 수 있습니다. 다만, 자신이 조작하지 않는 카메라를 사용할 때는
스튜디오 모드와 같이 Sub Screen 들을 동시에 모니터링 하지 않으면 적합한 카메라 시점을 송출하기 어렵습니다.
상황에 따라 잘 사용하시기 바랍니다.
