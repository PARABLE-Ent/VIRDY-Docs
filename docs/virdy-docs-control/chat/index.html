채팅

VIRDY 월드에서 채팅을 입력할 수 있습니다.

전체 채팅​

Enter 키 입력시 위와 같이 전체 채팅으로 입력창이 활성화 됩니다.

역할 채팅 (팀 채팅)​

Y 키를 입력하면 역할 채팅으로 활성화 됩니다.

매니저 유저가 Y 키를 입력하여 채팅을 보내면 매니저 유저에게만 채팅이 보입니다.
마찬가지로 크리에이터 유저가 Y 키를 입력하여 채팅을 입력하면 크리에이터 유저에게만 보입니다.
