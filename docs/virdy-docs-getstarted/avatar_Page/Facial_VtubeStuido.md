# VTubeStudio 세팅

:::caution
**VtubeStudio**의 연결 방식은 iFacialMocap, FaceMotion3D와 비슷합니다. <br/>
다만, 현재 공식 가이드는 제작중에 있으므로 특별한 사유를 제외하고는 <a href="./Facial_iFacialMocap" class="custom-link">**iFacialMocap/FaceMotion3D**</a> 사용을 권장드립니다.
:::
