iFacialMocap 세팅

VIRDY는 iFacialMocap 앱과 동일한 개발사의 제품인 FaceMotion3D를 사용할 수 있습니다.

tip

페이셜 기기로 사용하는 iPhone의 경우, 12 이상의 기기에서 최상의 품질을 보여줍니다.
12 미만의 버전에서도 작동은 하나 페이셜 품질이 다소 낮을 수 있습니다.

설정​


1. 활성화를 On으로 변경합니다.

2. 사용할 iPhone과 PC를 같은 네트워크로 연결합니다.

3. iPhone의 IP 주소를 디바이스 아이피 주소에 입력하거나, 페이셜 앱에 PC IP를 입력합니다.
PC IP 주소는 아래 내 컴퓨터 아이피 주소 의 눈 모양 아이콘을 클릭하여 확인할 수 있습니다.

4. 포트를 페이셜 앱의 포트 넘버와 동일하게 설정합니다.

정상적으로 세팅했는데 연결이 안돼요 !

위 과정을 정상적으로 진행했음에도 페이셜 연결이 안되었다면,
VIRDY 프로그램의 방화벽을 허용하여야 합니다.


윈도우 검색창에서 'Windows 방화벽에서 앱 허용' 을 입력 후 실행합니다.
'설정 변경' 을 클릭 후 'virdy' 를 찾아 전부 활성화 해주세요.

위 과정을 거쳤는데도 연결이 되지 않는다면, PC와 iPhone이 동일한 네트워크인지 확인하여주세요.
